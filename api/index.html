Securly PHP Question API List

POST BlockedList.php
Get the Blocked List
Parameters
  • page: Page of display list
  • many: How many per page
  • Example: BlockedList.php?page=1&many=5
    • Return { "StatusCode": "200", // Status Code (200, 404) "Total": 5, // Total numbers of data "List": [] // Data Array }
    POST BlockedAdd.php
    Add a Blocked URL
    Parameters
  • url: The URL
  • Example: BlockedAdd.php?url=http://google.com
    • Return { "StatusCode": "200", // Status Code (200, 404) "Message": "XXX Added" // Messages }
    POST BlockedRemove.php
    Remove Blocked URL
    Parameters
  • blocked_id: The Blocked URL's ID
  • Example: BlockedRemove.php?blocked_id=1
    • Return { "StatusCode": "200", // Status Code (200, 404) "Message": "XXX Removed" // Messages }
    POST OpenURLCheck.php
    Check the URL is Blocked or not
    Parameters
  • url: The URL
  • Example: OpenURLCheck.php?url=http://www.google.com
    • Return { "StatusCode": "200", // Status Code (200, 404) "Message": "SAFE" // Messages }
